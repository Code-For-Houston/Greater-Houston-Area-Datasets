# Unofficial Greater Houston Area Data Portal

## What is the point?
The goal of this README is to help connect individuals to the many different websites for datasets about the Greater Houston Area. 

## List of Data Portals

- City Data Portals
	- [City of Houston Data Portal](https://cohgis-mycity.opendata.arcgis.com/)
	- [Houston Public Works Open Data Portal](https://geohub.houstontx.gov/)

- County Data Portals
	- [Harris County Data Portal](https://geo-harriscounty.opendata.arcgis.com/)
	- [Montgomery County Data Portal](https://data-moco.opendata.arcgis.com/)
	- [H-GAC Data Portal](https://gishub-h-gac.hub.arcgis.com/)

- State Data Portals
